Django template engine

  • Blocks

    Defines a block that can be overridden by child templates.

  • For ... in

    Loops over each item in an array, making the item available in a context variable.

  • If

    The if tag evaluates a variable, and if that variable is “true” (i.e. exists, is not empty, and is not a false boolean value) the contents of the block are output.

    The if tag may take one or several elif clauses, as well as an else clause that will be displayed if all previous conditions fail. These clauses are optional.

  • Variables

    When the template engine encounters a variable, it evaluates that variable and replaces it with the result. Variable names consist of any combination of alphanumeric characters and the underscore ("_") but may not start with an underscore, and may not be a number. The dot (".") also appears in variable sections, although that has a special meaning. Importantly, you cannot have spaces or punctuation characters in variable names.
